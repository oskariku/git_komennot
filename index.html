Git-peruskomennot

Komennot, jotka tehdään alussa

  • git init - Create an empty Git repository or reinitialize an existing one
  • git remote add origin git@github.com:oskariku/git_komennot.git -
  • Ensimmäisellä kerralla: git push -u origin master -

Yleisimmät komennot

  • git add . -
  • git commit -m "tekstiä" -
  • git push -

Muista tehdä README.md ja .gitignore
